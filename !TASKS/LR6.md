Лабораторная работа №6
======================

Класс «Матрица»
---------------

Составить описание класса для объектов прямоугольных матриц,
задаваемых массивом вещественных чисел типа **double**.
Компоненты матрицы должны быть инкапсулированы.

I. Конструкторы
---------------

Предусмотреть:

- а) по умолчанию (пустая матрица);
- б) квадратная матрица заданного размера;
- в) прямоугольная матрица заданных размеров;
- г) инициализация из двумерного массива;
- д) копирование.

В конструкторах и деструкторе выводить диагностические сообщения.

II. Методы класса
-----------------

1) проверка возможности умножения матриц (static);
2) проверка возможности сложения матриц (static);
3) поиск максимального элемента;
4) поиск минимального элемента.

III. Операторы
--------------

Перегрузить операторы:

- сложения;
- вычитания;
- умножения матриц;
- умножения на скаляр.

IV. Индексация и свойства
-------------------------

Обеспечить доступ к элементам по индексу строки и столбца.
Предоставить доступ (только чтение) к количеству строк и столбцов.

V. Представление
----------------

Перегрузить метод `ToString()` для построчного представления матрицы
с выравниванием столбцов.

При невозможности выполнения операций генерировать исключения.
