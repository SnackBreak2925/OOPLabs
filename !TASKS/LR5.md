Лабораторная работа №5
======================

Цель работы
-----------

Изучение механизма применения шаблонов функций и классов.

Задание к лабораторной работе
-----------------------------

Работа базируется на результатах лабораторных работ №2–4.

### 1. Шаблонный класс матрицы

Создать шаблонный класс матрицы и протестировать его с типами:

- базовые типы C (`int`, `double`);
- класс матрицы из ЛР№2;
- класс дроби из ЛР№4;
- матрица, параметризованная другой матрицей.

При необходимости использовать `typedef` для вложенных шаблонов.

### 2. Полиномиальная дробь

Создать класс для представления полиномиальной дроби.

Обеспечить:

- арифметические операции;
- автоматическое сокращение;
- декомпозицию неправильной дроби.

### 3. Полином с рациональными коэффициентами

Коэффициенты — обыкновенные дроби.

Обеспечить:

- арифметические операции;
- деление с остатком;
- удаление нулевых коэффициентов;
- вычисление значения полинома.
