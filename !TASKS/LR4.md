Лабораторная работа №4
======================

Цель работы
-----------

Создание классов математических объектов с перегрузкой операций.

Задание к лабораторной работе
-----------------------------

### 1. Класс «Обыкновенная дробь»

Составить описание класса для представления дроби вида `A / B`, где
`A` и `B` — целые числа.

Обеспечить:

- выполнение арифметических операций (`+`, `-`, `*`, `/` и составные);
- автоматическое сокращение дроби;
- декомпозицию неправильной дроби на целую часть и правильную дробь.

### 2. Класс «Полином»

Полином вида степени `n` с вещественными коэффициентами.

Обеспечить:

- выполнение операций `+`, `-`, `*`;
- деление полиномов с остатком (алгоритм Евклида);
- удаление старших членов с нулевыми коэффициентами;
- перегрузку оператора `()` для вычисления значения полинома.

Для всех типов реализовать перегруженный оператор присваивания.

*УКАЗАНИЕ*: арифметические операторы реализовать через дружественные
функции.
